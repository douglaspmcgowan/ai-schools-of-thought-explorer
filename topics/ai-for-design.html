AI for Design

How designers actually partner with AI. Three schools: AI as a hands-on design tool, prose-first interfaces that treat text as the substrate, and the career/team argument about what design work becomes.

What's actually being argued. Not "will AI replace designers" — that framing is exhausted. The live questions are: which parts of the craft move to the model, which parts move to prose, and how teams restructure when both answers keep shifting.

AI as design tool

Image and code models used hands-on in the design loop — Nano Banana for UI, Claude Code in Figma. The concrete, demo-able version of AI-assisted design.

Prose-first interfaces

If AI runs on text, the interface should too. Design shifts from pixels-first to prompt-first; spec and prose become the primary artifacts.
