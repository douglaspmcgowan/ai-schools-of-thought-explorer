Constitutionalism critiques

The "constitution" metaphor is a category mistake: these aren't laws, there's no accountability, and the analogy smuggles legitimacy it hasn't earned. Multiple critiques from different angles converge on the same objection.

Why this matters. If the constitutional frame becomes the default language for AI alignment, it shapes how courts, regulators, and the public read the labs' commitments. The critics are warning about the downstream effects of borrowed legitimacy.

The argument

Lawfare's "The Code Is Not the Law" is the constitutional-law critique: real constitutions have enforcement, interpretation, and amendment procedures; Claude's does not. Using the word "constitution" borrows legitimacy without the accompanying obligations. Nintil's "On Claude's Constitution" is the anthropomorphization critique: talking about a model's "values" and "reasoning" with the same syntax we use for humans is a category error that will generate specific policy mistakes. Windows on Theory's "Thoughts on Claude's Constitution" is the most sympathetic-but-pointed technical critique — taking the project seriously while noting specific failure modes.
