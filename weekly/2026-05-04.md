## Techniques & tricks

**Steerable thinking plans** (https://sureprompts.com/blog/advanced-prompt-engineering-2026-claude-gpt5-gemini): Add "Before you begin, outline your step-by-step plan for this task. Wait for my approval before proceeding" to any prompt, letting you redirect the model's approach before it does the expensive work.

**Control reasoning via API parameters, not prompts** (https://www.lakera.ai/blog/prompt-engineering-guide): On 2026 frontier reasoning models (Claude 4.6, GPT-5.4, Gemini 2.5), "think step by step" is counterproductive — set reasoning effort through API parameters instead, producing faster and more predictable outputs.

**AI memory with biological decay — 52% recall** (https://news.ycombinator.com/item?id=47914367): Applying the Ebbinghaus forgetting curve to AI context management — memories decay on a schedule rather than by token count — achieving 52% recall in long-running agent sessions.
