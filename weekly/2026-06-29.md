## Techniques & tricks

**Loop Engineering** ([lushbinary.com/blog/loop-engineering-ai-coding-agents-guide](https://lushbinary.com/blog/loop-engineering-ai-coding-agents-guide/)): The 2026 reframe of AI coding: instead of typing prompts by hand, write the program that prompts the agent for you — run it in a `while` loop against a written spec, let it pick and implement one task per iteration, restart fresh each round; pioneered by Geoffrey Huntley with Claude Code, popularized this month by Addy Osmani and Anthropic's Boris Cherny.

**Specs-first + AI-on-AI review** ([addyo.substack.com/p/my-llm-coding-workflow-going-into](https://addyo.substack.com/p/my-llm-coding-workflow-going-into)): Addy Osmani's practitioner workflow — write a detailed spec *before* prompting, break work into small iterative chunks, then run a second AI model over the first AI's output as an automated quality gate; produces code that survives real review rather than "looks right" one-shots.

**Loops for non-coders** ([every.to/context-window/loops-for-non-coders](https://every.to/context-window/loops-for-non-coders)): Every.to's Austin Tedesco on how business users in legal, finance, and sales are adopting loop-based AI workflows to build working apps without writing code, dissolving the developer/non-developer distinction in practice.
